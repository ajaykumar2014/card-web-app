# Card-web-app

## Checkout the code branch 

 $ git checkout https://github.com/ajaykumar2014/card-web-app.git
 
## Build and Run App by Docker-composer
  $cd card-web-app/
  
  $git checkout master
  
  $git pull
  
  $ ./docker-build.sh 
  
 
## WEB view access

  Open Browser and paste - http://localhost:3001/
